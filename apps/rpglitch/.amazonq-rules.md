# RPGlitch Folder Rules

## UI Standards
- Text labels required, no icon-only buttons
- Minimalist aesthetic with tiny text where appropriate
- Perchance-compatible HTML structure

## Code Standards  
- Use Cash DOM for DOM manipulation
- Dexie.js for IndexedDB operations
- Modern ES2023+ features
- Zero technical debt implementations

## Build Requirements
- Single-file output for Perchance
- Inline all external dependencies
- SCSS compilation to CSS
- Minification for production
