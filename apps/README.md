# Applications

This directory contains the primary, user-facing web applications of the JooduG-default repository.

For a detailed overview of the "Perchance Framework" used to build these applications, please see the [Perchance Framework Overview](../docs/guides/perchance-framework-overview.md).

- **`/imageglitch`**: A single-file web application for applying generative glitch art effects to images.
- **`/rpglitch`**: A multi-file web application for creating, managing, and viewing entities for role-playing games.
