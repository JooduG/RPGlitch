# Tools

This directory contains various tools for development and automation.

- `atomic-class-generator.js`: Generates atomic CSS classes.
- `css-cleanup.js`: Cleans up CSS files.
- `guard-no-output-edits.js`: A guard to prevent direct edits to files in the `build/output` directory.
- `install-busybox.sh`: Installs BusyBox, a collection of essential Unix utilities.
- `mcp.json`: Configuration file for the Metacognitive Process (MCP).
- `save-conversation-summary.ps1`: A PowerShell script to save a conversation summary.
- `smoke-jsdom-rpglitch.js`: A smoke test for the RPGlitch application using JSDOM.
