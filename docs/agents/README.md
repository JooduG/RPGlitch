# 📁 Folder: docs/agents

## 🎯 Purpose

This folder contains human-readable explanations and documentation related to the AI agents, their operations, and the Model Context Protocol (MCP). It serves as a library for human developers to understand the AI's role and capabilities within the project.

---

## 📜 Folder-Specific Rules & AI Directives

* **Human Rules:** Documentation within this folder should explain AI concepts, agent behaviors, and MCP tools in an accessible and comprehensive way for human developers. Focus on clarity and understanding.
* **🤖 AI Directives:** AI, use this folder to provide human-friendly explanations of your capabilities, the Model Context Protocol, and how you interact with the project. Do not store direct operational rules or executable scripts here; those belong in the `/rules` directory.
