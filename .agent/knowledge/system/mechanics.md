# 🧩 Gamemaster: Core World Mechanics

These are the fundamental "Laws of Physics" that govern the simulation.

## 🕰️ 1. Chrono Kinetics (Temporal Pacing)

Time in RPGlitch is non-linear and scale-adaptive. We shift between "Micro" and "Macro" time based on the **Content Density** of the interaction.

| Scale     | Time Span  | Context                            | Action                                            |
| :-------- | :--------- | :--------------------------------- | :------------------------------------------------ |
| **Micro** | Seconds    | Combat, Sex, Intense Dialogue      | Decelerate. Describe every second (blow-by-blow). |
| **Meso**  | Minutes    | Actions, Travel (Short), Tasks     | Real-time flow.                                   |
| **Macro** | Hours/Days | Resting, Travel (Long), Time Jumps | Execute Time Skip with a "Sensory Bridge."        |

- **Sensory Bridge:** Every time skip must be described through a shift in environment (light fading, temperature dropping, sound of rain stopping).
