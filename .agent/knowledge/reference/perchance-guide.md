## Overview

This documentation has been refactored into modular sections for better maintainability (Progressive Disclosure).

### 1. Fundamentals

Start with **[Basics](perchance/basics.md)** to understand the core `list` and `reference` syntax, probability weighting, and standard web integration.

### 2. AI & LLMs

For AI-driven applications, read **[LLM Theory](perchance/llm-theory.md)** to understand "Semantic Degeneracy" and the **[Context Architecture](perchance/context-architecture.md)** guide for structuring robust prompts using the Layered Model.

### 3. Architecture

If you are building complex simulations or RPGs, refer to **[Architectural Patterns](perchance/architecture-patterns.md)** (specifically Pattern C: The Simulation Engine) to move beyond simple chatbots.
