# 🧭 Mesmer Nexus: The Design OS

> **Role:** The "Global Brain" for RPGlitch Aesthetics.
> **Pillar:** Mind Control (The Mesmer).
> **Objective:** Bridge the gap between Lore, Design, and Code.

## 1. The Core Directives

All design work must align with the **Genre-Agnostic Minimalism** aesthetic. If the UI feels cluttered or overly specific, it is a failure of the Illusion.

- **[Master Design Guide](../../design.md)**: The executive summary for AI prompting.
- **[Aesthetic Inspiration](./inspiration.md)**: Visual mood boards and narrative motifs.
- **[Quality Standards](./quality-gate.md)**: The strict gate for production-ready UI.

## 2. Technical Mapping

The nexus connects descriptive tokens to actual code implementation.

- **Variables**: [src/mesmer/scss/abstracts/\_variables.scss](../../../src/mesmer/scss/abstracts/_variables.scss)
- **Mixins**: [src/mesmer/scss/abstracts/\_mixins.scss](../../../src/mesmer/scss/abstracts/_mixins.scss)

## 3. The Design Baton Workflow

1.  **Draft**: Start in [ideation.md](../nexus/vision/ideation.md).
2.  **Prototype**: Use [Stitch Loop](../../skills/mesmer/knowledge/stitch-iteration.md).
3.  **Synthesize**: Update [DESIGN.md](../../design.md) with new tokens.
4.  **Audit**: Run `/review-design` against the [Quality Gate](./quality-gate.md).
