# 📖 Lexicon (The Dictionary)

> **Red Thread:** Consistent terminology ensures unified orchestration between AI skills and project vision.

## 🏛️ The Skill Matrix (Architecture)

1. **🕹️ Engine**: Narrative progression, time flow, and logic ticks (`src/core/engine/`).
2. **🛠️ UI**: Svelte components and interactive layout (`src/ui/`).
3. **🎭 Polish**: Visuals, Audio, and Atmosphere (`src/theme/`, `src/theme/audio`).
4. **📚 Data**: Persistence and State Management (`src/data/`).
5. **🛡️ Security**: Validation, Sanitization, and Technical Audit (`src/core/security/`).
