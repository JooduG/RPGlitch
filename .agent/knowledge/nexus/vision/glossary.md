# 📖 Domain Language

## The Five Pillars

- **Gamemaster:** The Central Logic & Time Orchestrator.
- **Artificer:** The Structural Body (UI Components, Layouts, CSS).
- **Mesmer:** The Sensory System (Voice, SFX, Visuals).
- **Scholar:** The Memory & Database (Dexie/Echo).
- **Warden:** The Security & Physics Enforcer.

## Core Concepts

- **Monolith:** The single [`index.html`](../../../../dist/index.html) output artifact.
- **Runes:** Svelte 5 reactivity system (`$state`, `$derived`, `$effect`).
- **Chrono**: The Turn-Based Time Machine ([chrono.svelte.js](../../../../src/gamemaster/chrono.svelte.js)).
- **Echo:** The Database Repository Pattern.
- **Fractal:** A generated world/setting.
- **Chin:** The mobile bottom drawer UI.
- **Signature Color:** The dynamic theme color assigned to a Character or Entity.
