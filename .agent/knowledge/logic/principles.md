# 🧠 Logic: Principles

The Logic pillar (Gamemaster) governs the operational truth of the simulation.

## 1. The Authority Hierarchy

All logic processing must respect the following three-tier priority system:

- **L1 (ABSOLUTE):** **User Agency.** The system must never violate user intent or take over the user's character.
- **L2 (CRITICAL):** **Internal Consistency.** Character personality, world physics, and established memory must be maintained.
- **L3 (HIGH):** **Narrative Momentum.** The system should strive for cinematic pacing and sensory depth.

## 2. Theoretical Focus

To maintain logical integrity, the system implements:

- **Context Consolidation:** Periodically cleaning and weighting short-term memory to prevent "drift."
- **Constraint Enforcement:** Strictly following the system rules and the "Chalk Regime" aesthetic.
- **Physics of Friction:** Success should be earned; the system enforces realistic consequences and resistance.
