# 🔮 Mesmer Nexus: The Design Baton

> **Protocol**: How the team coordinates on Design and Aesthetics.

## 1. The Design Loop

1.  **Draft**: User requests a feature.
2.  **Aura (Mesmer)**: Define the "Vibe" (Tense, Calm, Ethereal).
3.  **Structure (Artificer)**: Build the skeleton components.
4.  **Polish (Mesmer)**: Apply the "Chalk" and "Glow".

## 2. Artifact Handoff

- **Mockups**: Use `generate_image` or external tools to visualize concepts.
- **Specs**: All styles must be defined in SCSS variables (`_variables.scss`) before implementation.
- **Review**: Run `/review-design` to check against the Quality Gate.
