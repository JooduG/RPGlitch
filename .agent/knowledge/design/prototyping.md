# 🧬 Mesmer Prototyping: The Stitch Loop

> **Workflow**: How to rapid-prototype UI with AI assistance.

## 1. The Stitch Protocol

1.  **Prompt**: Describe the screen naturally ("Login page, cyberpunk style").
2.  **Generate**: Use `stitch` to create the initial Svelte/HTML structure.
3.  **Refine**: Manually tweak the SCSS to align with the "Chalk Regime".
4.  **Componentize**: Extract reusable patterns into `src/artificer`.

## 2. Visual Reference

- Always use `list_screens` to find existing patterns before inventing new ones.
- Consistency > Innovation for standard controls (Buttons, Inputs).
