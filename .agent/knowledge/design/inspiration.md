# 🌌 Mesmer Inspiration: The Illusory Rules

> **Philosophy**: "The UI is a Silent Stage. It breathes with the narrative but never competes with it."

## 1. Minimalist Resonance

- **Neural Minimalism**: The interface should feel like a neural extension, not a tool.
- **Narrative Resonance**: Colors and "Aura" shift subtly to match the emotional state of the story.
- **The "Chalk" Aesthetic**: Use `#222326` (Chalk) instead of pure black for deep surfaces to avoid eye strain and create a matte, premium feel.

## 2. Dimensionality & Physics

- **Soft Depth**: Use layered shadows (`box-shadow`) to create elevation.
- **Glass**: Use `backdrop-filter: blur(12px)` for overlays, but avoid the "frosted shower door" look.
- **Weight**: Elements should feel physically grounded. When they move, they follow kinetic physics (elastic, snappy).
