# 🎨 Lab 02: Interaction (Sensory & Immersion)

> **Red Thread:** Pushing the boundaries of sensory delivery through kinetic UI and atmospheric depth.

## 🖼️ Reactive Visuals

Speculative concepts for dynamic character and environment representation.

- **Reactive Triptychs:** A 3-panel portrait layout for characters that shifts based on state.
    - **Persona:** Base visual appearance.
    - **State:** Real-time emotional overlays (e.g., glitching, saturation shifts).
    - **Atmosphere:** Environment lighting and color sync.
- **Cinematic Cutaways:** Triggering vignettes from an NPC's perspective (e.g., `[MEANWHILE]` blocks) to show off-screen events.

## 🔊 Atmospheric Audio

Concepts for deeper auditory immersion managed by the **🎭 Polish** skill.

- **Resonance Loops:** Dynamic contextual audio that syncs with narrative headers and location shifts.
- **Micro-SFX:** Context-aware audio cues for system events, UI interactions, and engine checks (e.g., "Entropy Click").

## 🌊 Kinetic UI

Speculative interaction models for the user interface.

- **Glitch Haptics:** Simulating "mechanical failure" or "digital decay" through CSS-based visual feedback.
- **Depth Layering:** Using shadows and 2% grain texture to provide a tactile feel to the index.html interface.
