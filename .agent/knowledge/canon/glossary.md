# 📖 Project Lexicon (Glossary)

> **Red Thread:** Consistent terminology ensures unified orchestration between AI skills and project vision.

## 🏛️ The Five Pillars (Architecture)

1. **🕹️ GameMaster (The Engine):** Logic execution, time progression (`src/core/engine/`).
2. **🛠️ Artificer (The UI):** Svelte components, layout (`src/ui/`).
3. **🎭 Mesmer (The Senses):** Visuals, Audio, Atmosphere (`src/theme/`, `src/media/`).
4. **📚 Scholar (The Memory):** Persistence and State Management (`src/data/`).
5. **🛡️ Warden (The Protector):** Security, Validation, and Testing (`src/core/security/`).

## 🧩 Core Concepts

| Term                  | Definition                                                             |
| :-------------------- | :--------------------------------------------------------------------- |
| **Chrono Kinetics**   | Time progression system based on input density (Micro/Macro).          |
| **Entropy Engine**    | Randomizer system for "Fail-Forward" logic injection.                  |
| **Resonance Monitor** | HUD tracking Trust, Will, and Tension variables.                       |
| **Lorebook (ANEX)**   | Speculative knowledge injection system.                                |
| **Silent Stage**      | Design philosophy: Minimize UI noise, maximize text immersion.         |
| **Echo**              | A persistent memory fragment stored by the Scholar.                    |
| **Fractal**           | Speculative branching narrative path.                                  |
| **DevMode**           | Technical analysis and state inspection interface.                     |
| **StoryMode**         | Primary narrative and user interaction interface.                      |
| **GM Mode**           | Narrative control interface for the plot/world. NEVER "Director Mode". |
| **Runes**             | Svelte 5 reactive primitives used for state management.                |
