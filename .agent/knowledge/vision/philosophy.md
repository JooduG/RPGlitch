# 👁️ Mesmer: Core Philosophy

> **Directive:** "Genre-Agnostic Minimalism". The UI is a canvas, not a painting.

RPGlitch is built on the friction between high-order simulation logic and a stark, atmospheric sensory layer. We reject the "Game UI" tropes of buttons, bars, and neon. Instead, we embrace the **Neural Baseline**.

## 🧠 1. Neural Minimalism

Modern interfaces demand attention. RPGlitch _recedes_ from it.

- **The Silent Stage:** The interface should feel like an empty theater waiting for the first word. Darkness is the default; light is a narrative event.
- **Cognitive Recoil:** We avoid "Information Overload." If a piece of data isn't vital to the current narrative beat, it is hidden or pushed to the peripheral HUD (Resonance Monitor).
- **Subtext over UI:** We prioritize describing the _feeling_ of a menu over rendering a complex table.

## 🎨 2. The Chalk Regime

The visual language is defined by the **Chalk Regime**—a system of high-contrast, atmospheric surfaces.

- **Primary Surface:** `#222326` (Chalk). A deep, textured gray that feels like a blackboard or a stone slab.
- **Accents:** We do not use "Brand Colors." Instead, we use **Narrative Tinting**. If the scene is cold, the shadows shift to desaturated blue; if it's violent, a hint of oxblood red.
- **Soft Depth:** We reject flat design. We use layered shadows, slight noise textures, and `backdrop-filter: blur()` to create a sense of physical space.

## ⚖️ 3. Authority Hierarchy

All decisions in the system follow a strict order of operations:

1. **L1_ABSOLUTE (User Agency):** The user's choices are the external, immutable force that triggers the world.
2. **L2_CRITICAL (Integrity):** Character consistency, physical laws, and the persistent memory of the simulation.
3. **L3_HIGH (Momentum):** Narrative pacing, sensory depth, and cinematic flair.

## 🛰️ 4. The Reactive Core

Everything in RPGlitch is reactive. State drives Reality. We do not use "Glue Code"; we use **Runes** and **Resonance**. Every action by the user triggers a calculation in the **Gamemaster**, which ripples through the **Mesmer** (visuals) and is anchored by the **Scholar** (memory).
