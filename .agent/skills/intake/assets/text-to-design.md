# Workflow: Text to Design (Aesthetic Scoping)

1. **Analyze Intent**: What is the "vibe" or visual atmosphere the user is requesting?
2. **Translate to Tokens**: Map the visual requests to `CHALK_REGIME` tokens using the **Designer**'s logic.
3. **Draft Aesthetic Spec**: Document the core color palette, typography, and surface elevations.
4. **Handoff to Designer**: Pass the aesthetic specification to the **Designer** to orchestrate the implementation across AV/CSS skills.
