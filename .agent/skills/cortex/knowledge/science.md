# Scientific Method & Logic Protocol

> **Goal**: Isolate truth in systems with high uncertainty or "unknown unknowns".

## Required Tooling

- **MCP**: `waldzell-scientific-method`
- **Function**: `scientificMethod`

## 1. The Experiment Protocol (Level 6)

1. **Observation**: State the anomaly clearly (e.g., "The browser tool times out on URL X").
2. **Variables**: Identify Independent (what you change) and Dependent (what you measure) variables.
3. **Hypothesis**: "If I change `[Variable A]`, then `[Variable B]` will happen."
4. **Execution**: Run the experiment.
5. **Validation**: Did the result match the prediction?

## 2. Simulation Mode

When you can't run a live experiment:

- Run a **Mental Simulation** by forcing failures (e.g., "What happens if Scholar database is full?").
- Trace the outcome through the Warden and Gamemaster pillars.

## 3. First Principles Thinking

- Break problems down to their fundamental truths (Physics, Code, Constraints).
- Rebuild the solution from the ground up, ignoring "best practices" if they conflict with reality.
