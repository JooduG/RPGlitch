# Cortex: Science Layer (L6)

> **Goal**: Isolate truth in systems with high uncertainty or "unknown unknowns".

## The Protocol: The Experiment

1. **Observation**: State the anomaly clearly (e.g., "The browser tool times out on URL X").
2. **Variables**: Identify Independent (what you change) and Dependent (what you measure) variables.
3. **Hypothesis**: "If I change [Variable A], then [Variable B] will happen."
4. **Execution**: Run the experiment.
5. **Validation**: Did the result match the prediction?

## Simulation Mode

When you can't run a live experiment, run a mental simulation by forcing failures (e.g., "What happens if Scholar database is full?"). Trace the outcome through the Warden and Gamemaster pillars.
