# 🧠 Skill: Cortex (The Mind)

The Cortex is the single entry point for all non-trivial reasoning. It replaces fragmented skills with a unified **Complexity Router**.

## 1. The Complexity Matrix

| Level  | State        | Intent                        | Protocol              | Knowledge Layer                                      |
| :----- | :----------- | :---------------------------- | :-------------------- | :--------------------------------------------------- |
| **L1** | **Reflex**   | Simple edits, renames, typos. | Just Code             | -                                                    |
| **L2** | **Logic**    | Multi-file changes, planning. | `sequential-thinking` | [planning.md](./knowledge/planning.md)               |
| **L3** | **Debug**    | Bug fixes, error loops.       | `debugging-strategy`  | [debugging-strategy](../debugging-strategy/SKILL.md) |
| **L4** | **Reframe**  | Stuck, suspected XY problem.  | `mental-modeling`     | [reframe.md](./knowledge/reframe.md)                 |
| **L5** | **Critique** | High accuracy/Second opinion. | `council-debate`      | [critique.md](./knowledge/critique.md)               |
| **L6** | **Science**  | Performance tuning, unknowns. | `experiment`          | [science.md](./knowledge/science.md)                 |

## 2. Universal Protocol

1. **Activate**: When a task is assigned, identify the Complexity Level.
2. **Audit (Metacognition)**: Invoke the [Audit Layer](./knowledge/metacognition.md) if you encounter loops or friction.
3. **Switch Mode**: Transition your thought process to the corresponding Pillar.
4. **Reference Knowledge**: Use the specialized layers in `./knowledge/` to deepen the analysis.
