## 🎯 Strategic Context

- **Swarm Sovereignty**: Evaluates the "Parallel Win." A swarm is sent for approval if:
    - Task estimation > 20 minutes total (10m per parallel branch).
    - Sub-tasks touch independent files.
    - Parallel execution provides a measurable velocity gain.
- **Pipeline Architecture**: Enforces the `Acquire → Prepare → Process → Parse → Render` sequence. Each stage must be discrete, idempotent, and grounded in the file system state.
- **Cognitive Routing**: Trigger `waldzell-clear-thought` for architectural reframing and `mcp-sequentialthinking-tools` for complex logic chains or any of the other cognitive models.

## 📋 Procedure

### 1. Strategic Discovery & Triage
- **Law Review**: Analyze the task against the 6 Axiomatic Laws ([Foundation](../../rules/01-foundation.md), [Simulation](../../rules/02-simulation.md), [Infrastructure](../../rules/03-infrastructure.md), [Aesthetics](../../rules/04-aesthetics.md), [Intelligence](../../rules/05-intelligence.md), [Compliance](../../rules/06-compliance.md)).
- **Task-Model Fit**: Evaluate if the task requires high-level synthesis (AI) or precise computation (Classic Code).
- **Ambiguity Resolution**: Invoke [Intake](../../skills/intake/SKILL.md) if user intent is unclear or ambiguous.

### 2. Architectural [Topography](./references/topology-overview.md)
- **File System State Machine**: Map the project topography. Mandate that state is tracked via directory structure and file existence (e.g., `parsed.json` existence gates the render stage).
- **Context Isolation**: Ensure specialized agents in a swarm have strict file-level boundaries to prevent collision.
- **Staged Foundation**: For complex features, architect in this sequence:
    1. **Data layer** (Dexie schema, Migrations).
    2. **Logic layer** (Engine mutations, Security boundaries, JavaScript).
    3. **UI layer** (Svelte 5 Components, CSS, Motion, Audio).

### 3. Strategizing & Meta-Memory
- **Architecture Decision Framework (ADF)**: Capture rationale and trade-offs for significant architectural decisions in the implementation plan.
- **The confidence gate**: Enforce a collective audit of swarm outputs before merging.

## 🚫 Anti-Patterns

- **Monolithic Pipelines**: Combining unrelated stages into a single execution turn.
- **Strategic Drift**: Violating [Foundation](../../rules/01-foundation.md) or the local "Sovereign State" topography.
- **Identity Drift**: Sub-agents in a swarm losing their specialized purpose.
