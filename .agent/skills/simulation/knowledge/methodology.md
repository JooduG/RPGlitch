# 🏗️ Design & Build Methodology

> **Skill:** simulation
> **Red Thread:** The playbook for building and refining the sensory layer.

## 🏃 1. The Design-Baton Loop

1. **Draft (Scholar):** Define data requirements.
2. **Aura (Mesmer):** Define the visual mood.
3. **Structure (Artificer):** Build semantic HTML/layout.
4. **Resonance (Gamemaster):** Connect Runes to state.
5. **Polish (Warden):** Quality Gate audit.

## ⚡ 2. The Stitch Protocol (Rapid Prototyping)

- **Isolated Scoping:** Build in a single file with inline SCSS.
- **Token Injection:** Use `var(--app-...)` tokens from the start.
- **Fail Fast:** Revert immediately if layout breaks.

## 🛡️ 3. The Quality Gate (Audit)

1. **Visual:** Chalk Regime compliance.
2. **Interaction:** Snappy Curve usage.
3. **Accessibility:** Unique IDs and high contrast.
4. **Adaptability:** Graceful mobile degradation.
