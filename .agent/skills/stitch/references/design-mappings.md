# Reference: Antigravity Design Mappings

When translating concepts, strictly map them to these physical realities:

- "Stateful variable" -> `let count = $state(0);`
- "Derived value" -> `let doubled = $derived(count * 2);`
- "Background color" -> `background: var(--chalk-surface);`
- "Text color" -> `color: var(--chalk-foreground);`
- "Subtle border" -> `border: 1px solid var(--chalk-border);`
- "Primary Button" -> `background: var(--chalk-accent); color: var(--chalk-surface);`
