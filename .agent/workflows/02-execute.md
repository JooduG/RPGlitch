## Anti-Patterns

- **Ad-hoc Utilities**: Using inline styles or one-off CSS instead of tokens.
- **Legacy Reactivity**: Using `export let` or `writable()` in Svelte 5.
- **Shadow Logic**: Managing state in the DOM instead of the context block.
