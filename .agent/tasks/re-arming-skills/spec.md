# Spec: Re-Arming (Skills Refinement)

## Goal

Finalize the agent's sensory and structural toolkit by refining the `scribe` (Architectural Auditor) and `visuals` (Asset Generation) skills.

## Context

RPGlitch requires high-fidelity visuals and rigid structural integrity. Current skills are fragmented. This phase "re-arms" the agent to build and polish with maximum efficiency.

## Acceptance Criteria

- [ ] `scribe` skill has a functional `audit` script for standards compliance.
- [ ] `visuals` skill is scaffolded with Perchance `text-to-image` integration guidelines.
- [ ] `polish` protocol is verified against a core panel (`ControlPanel` or `StoryPanel`).

## Architecture

- **Scribe Auditor**: Node.js script to scan for Anti-Patterns.
- **Visuals Skill**: Promp engineering guidelines and Perchance API hooks.
- **Polish Integrity**: Strict separation of logic and style.
