# 🏛️ Eternal (The Soul)

> **Role**: Application Vision & Product Dream
> **Status**: Sovereign / Grounded

RPGlitch is a high-fidelity, local-first simulation engine where **State Drives Reality**. It bridges creative roleplay and mechanical truth through a recursive intelligence kernel.

## 1. The Vision

The engine aims to provide a seamless, minimalist interface for deep narrative immersion. It is not just a game, but a framework for manifesting imagination through code.

### Core Tenets

- **Atmospheric Canvas**: Minimalist "Chalk Regime" aesthetics focus immersion on imagination.
- **Agentic Pacing**: Autonomous orchestration of narrative beats and state transitions.
- **Local Sovereignty**: Zero-latency, browser-resident memory (Dexie.js).

## 2. Technical Soul

- **Framework**: Svelte 5 (Runes exclusively).
- **Persistence**: Dexie.js (IndexedDB).
- **Build**: Vite 8 / Rolldown (Single-file).
- **Aesthetics**: Vanilla CSS (Nordic Collection).
- **UI Architecture**: Headless components (`bits-ui`) paired with Svelte 5 for interactive layers (Dialogs, Modals, Tooltips).
- **Motion**: Centralized View Transition logic via strictly guarded singletons.
- **Intelligence**: Context Broker orchestrates Fractal RAG memory, bridging `firecrawl-mcp` and other protocols into the kernel.

## 3. Product Dream

The "Eternal" state of RPGlitch is a system where the AI acts as a perfect dungeon master, the UI is an extension of the story, and the user's local machine is the sovereign host of their reality.
