# ARCHIVED: Atomic CSS Utilities (Historical)

This folder contains **historical tools and utilities** that were used for generating and managing atomic CSS classes during the **migration of CSS to a single-file architecture** for the RPGlitch project.

### Historical Purpose
- These tools were used to reduce CSS specificity conflicts and create a more maintainable CSS structure by consolidating atomic classes directly into the main stylesheet.

### How to Use (Historical)
- `generate-atomic-css.js`: Script used to generate atomic CSS utility classes.
- `css-migration-tool.js`: Used to analyze existing CSS for potential atomic class conversions.
- `atomic-utilities.css`: Previously generated CSS utility classes (now merged).
- `atomic-utilities-doc.md`: Previous documentation for utility classes (now deprecated).
- `css-migration-report.json`: Detailed report of past CSS migrations.
- `css-migration-suggestions.md`: Suggestions for converting existing CSS (now archived).
