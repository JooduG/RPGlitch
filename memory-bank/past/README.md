# Past - Archive of Completed Work

This directory contains an archive of completed tasks, historical context, and past decisions. It serves as a historical record of the project's evolution.

## Contents

The documents are loosely categorized by their subject matter.

### Migrations & Refactoring
*   `phase-1-completion-summary.md`: Summary of the resolution of the "Mode System vs Role System" conflict.
*   `readme-unified-rule-system-migration.md`: Strategic plan for migrating to a unified README-based rule system.
*   `basic-memory-structure-migration-2025-08-26.md`: Notes on the migration to the four-bucket memory structure (past, present, future, forever).

### Feature Summaries & Fixes
*   `lint-test-fixes-2025-08-25.md`: Summary of Stylelint and Jest test fixes.
*   `chin-button-sync-summary.md`: Summary of the chin close button implementation.
*   `chin-focus-gradient-alignment.md`: Notes on chin focus and background alignment fixes.
*   `chin-lockup-hardening-2025-08-27.md`: Details on fixes for UI lockups related to the "chin" panels.
*   `chin-sync-bg-profile.md`: Notes on syncing the chin button and background profile.
*   `chin-wiring-fix-summary.md`: Summary of rewiring the chin controls.
*   `rpglitch-single-file-fixes-summary.md`: Summary of various fixes for the RPGlitch single-file build.
*   `ci-safe-build-summary.md`: Notes on making the build and tests more stable for CI environments.

### MCP & Tooling Updates
*   `mcp-master-update-2025-08-22.md`: Log of an update to the master MCP configuration.
*   `mcp-config-refresh-2025-08-26.md`: Notes on refreshing the MCP configuration and environment variables.
*   `phase3-1-implementation-complete.md`: Summary of the AI-Driven Rule Selection System implementation.

### General Records
*   `CHANGES.md`: A log of miscellaneous changes.
*   `changelog-2025-08-25`: A specific daily changelog.
*   `completed.md`: A file intended to archive completed tasks from READMEs (currently empty).
*   `context-2025-08-25.md`: A snapshot of the project's context on that date.
*   `maintenance-run-2025-08-25.md`: Log of a routine maintenance run.
*   `todo-2025-08-25.md` & `chin-wiring-fix-todo.md`: Archived to-do lists.

### Test Logs
*   `/mcp-tests`: Contains raw, time-stamped logs from MCP server test runs. This directory can become very large and is primarily for deep debugging of the tooling infrastructure.
