# Memory Bank Folder Rules

## Organization Standards
- Strategic/Tactical/Operational folder structure
- Clear file naming with dates and context
- Archive old files, don't delete
- Use markdown for documentation

## Content Standards
- Concise, actionable content
- Clear headings and structure
- Link to related files
- Include context and rationale

## MCP Integration
- Use basic-memory MCP for knowledge management
- Maintain semantic relationships
- Update context regularly
