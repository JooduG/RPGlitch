# Memory Bank Structure

Basic Memory is organized into four top-level buckets:

- past/ — recent completions, summaries, and lessons
- present/ — active context, today’s notes, execution plans
- future/ — ideas, plans, and upcoming initiatives
- forever/ — durable references, policies, and canonical guides

Notes:

- Prefer adding new content directly under these buckets.
- Avoid using the deprecated `projects/` folder; it exists only for backward compatibility and will be removed.
