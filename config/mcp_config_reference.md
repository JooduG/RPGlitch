# MCP Configuration Reference

This file serves as a directive for all agents and processes to use the central MCP configuration file located at '.cursor/mcp.json' for server settings. This ensures consistency and ease of management across all operations within this project.

## Configuration Location
- **File Path**: .cursor/mcp.json

## Instructions
- All MCP server configurations must be sourced from the above file.
- Do not create or maintain separate copies of the configuration.
- Any updates to MCP settings should be reflected in '.cursor/mcp.json'.

## Current MCP Servers
The list of MCP servers is dynamic and can be updated in '.cursor/mcp.json'. Agents should refer to this file for the most current configuration of available servers.

This reference ensures that all agents use the same configuration source for MCP operations, regardless of changes in the number or type of MCP servers.
