# AI Rule Selection Tools

## Overview

AI-powered tools for intelligent rule selection and task classification.

## Tools

### context-analyzer.js

Analyzes context to determine appropriate rules and approaches.

### intelligent-rule-selector.js

Intelligently selects the most appropriate rules for given tasks.

### performance-optimizer.js

Optimizes performance based on AI analysis.

### task-classifier.js

Classifies tasks to determine the best approach.

## Usage

These tools work together to provide intelligent automation for development tasks.
