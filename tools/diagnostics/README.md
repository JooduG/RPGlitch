# Diagnostic Tools

## Overview

Tools for analyzing and optimizing the project's CSS and performance.

## Tools

### automation-collect-diagnostics.js

Automated collection of diagnostic information from the project.

### atomic-class-generator.js

Generates atomic CSS classes for utility-first styling.

### css-cleanup.js

Cleans up and optimizes CSS files.

### css-performance-analyzer.js

Analyzes CSS performance and provides optimization recommendations.

## Usage

Run tools individually as needed for specific analysis tasks.

## Reports

Generated reports are stored as JSON files in this directory.
