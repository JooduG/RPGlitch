# 📁 Folder: tools/diagnostics

## 🎯 Purpose
This directory contains specialized scripts and tools for diagnosing issues, debugging, and gathering system information within the project. These tools are designed to help identify the root causes of problems and provide actionable insights for resolution.

---

## 📜 Folder-Specific Rules & AI Directives

*   **Human Rules:** Diagnostic tools within this directory should be non-invasive, provide clear and actionable insights, and be easy to execute. They should not alter the system state unless explicitly designed to do so with proper warnings.
*   **🤖 AI Directives:** AI, when encountering errors, unexpected behavior, or when instructed to investigate a problem, you must utilize the tools in this directory to gather diagnostic information. Report findings clearly and concisely, referencing the specific tool used and its output.
