# 🧪 Incubator: Feature Roadmap Concepts

These are high-level feature concepts and speculative milestones that are not yet scheduled for the active development `roadmap.md`.

## 🎭 Visual & Sensory

- **Reactive Triptychs:** A 3-panel portrait spread (Setting | Actor A | Actor B) designed for widescreen cinematic immersion.
- **Ambient Soundscapes:** Dynamic, contextual audio loops (e.g., rain intensifying based on narrative tension) triggered by the Gamemaster.
- **Material Textures:** Enhancing image prompts with visceral specificity (subsurface scattering, grime, micro-lighting) to differentiate the "RPGlitch Look."

## 🧠 Intelligence & Memory

- **Lorebook (ANEX):** _Antigravity Narrative EXtension_. A structured RAG (Retrieval-Augmented Generation) system for deep world-building and persistent character memory.
- **Social Mesh:** Autonomous NPC-to-NPC relationships that evolve off-screen, creating a sense of a living world.
- **Multiverse Branching:** A visual node graph allowing users to save and explore "What If" diversions from the primary narrative timeline.

## 🕹️ Meta-Narrative

- **Glitch Protocol:** Narrative events where the engine or the "simulation" addresses the user directly, breaking the fourth wall for cosmic horror or dramatic effect.
